About the Project

LANCast is an academic Software Defined Networking (SDN) project designed to demonstrate multicast traffic optimization inside LAN environments. It integrates Java-based streaming with an SDN controller to intelligently manage packet forwarding, ensuring low latency and efficient bandwidth usage.
