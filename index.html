ğŸ“Œ Overview

LANCast is a Software-Defined Networking (SDN) based one-to-many screen broadcasting system designed for offline Local Area Networks (LAN). It enables a single broadcaster to stream their screen in real time to multiple receivers without requiring internet access.

  • Java for screen capture and streaming
  • UDP Multicast for one-to-many communication
  • Ryu SDN Controller for traffic optimization
  • OpenFlow-enabled switches (Mininet simulation)
